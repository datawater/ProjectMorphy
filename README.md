## Building
Depedencies Required for building:
- [Crystal](https://crystal-lang.org/)
- [Sqlite3](https://www.sqlite.org/)

Install the shards depedencies
```bash
shards install
```

Build (Dev)
```
make debug
```

Build (Release)
```
make release
```
